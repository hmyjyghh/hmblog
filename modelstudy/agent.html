    1. agent 基本概念

    寒梦

    # 1. agent 基本概念

    • LLM + Memory + plan 工具

    agent 的大脑 记忆功能，历史对话，多轮对话

    规划，让模型主动思考，下一步该干嘛，帮助agent 决定下一步该干嘛

    工具调用

    # 2. 为什么需要 agent？

    • LLM的训练不是实时的，无法获取最新的信息

    • 控制力弱，json输出格式，控制不好

    • 无法执行复杂任务，本身不具备逻辑规划，分步骤计算

    常见的3种Agent实现策略

    1. ReAct Prompting: 结合推理(Reasoning)与行动(Acting)的提示策略, 模型一边思考一边调用工具完成任务。
    2. Plan & Execute: 模型先规划整个任务流程(如多个步骤或工具调用顺序), 再逐步执行各个步骤。
    3. OpenAl Functions: 通过函数调用接口,让LLM能直接调用外部系统的函数(如API、数据库查询等), 增强其工具使用能力。

    # 使用ango 里面的 OpenAIEmbedder 报错

    Document processing error: Error code: 403 - {'error': {'code': 'unsupported_country_region_territory', 'message': 'Country, region, or territory not supported', 'param': None, 'type': 'request_forbidden'}}

    这个错误仍然是 OpenAIEmbedder 导致的！即使您已经将对话模型换成了 DeepSeek，但嵌入模型仍然在使用 OpenAI，而 OpenAI 在您所在地区不可用。

    解决方案

    # 方案1：使用本地嵌入模型（推荐）

    import streamlit as st
from agno.agent import Agent
from agno.knowledge.pdf import PDFKnowledgeBase, PDFReader
from agno.vectordb.qdrant import Qdrant
from agno.tools.duckduckgo import DuckDuckGoTools
from agno.models.deepseek import DeepSeek
# 移除 OpenAIEmbedder
import tempfile
import os
from agno.document.chunking.document import DocumentChunking

def init_qdrant():
    """Initialize Qdrant with local embedder"""
    if not all([st.session_state.qdrant_api_key, st.session_state.qdrant_url]):
        return None
    try:
        # 使用 Ollama 本地嵌入模型
        from agno.embedder.ollama import OllamaEmbedder
        
        vector_db = Qdrant(
            collection=COLLECTION_NAME,
            url=st.session_state.qdrant_url,
            api_key=st.session_state.qdrant_api_key,
            embedder=OllamaEmbedder(
                model="nomic-embed-text",  # 推荐的嵌入模型
                base_url="http://localhost:11434"  # Ollama 默认地址
            )
        )
        return vector_db
    except Exception as e:
        st.error(f"Ollama embedder failed: {str(e)}")
        return None

    # 方案3：修改侧边栏配置

    在您的 Streamlit 侧边栏中添加嵌入模型配置：

    with st.sidebar:
    st.header("🔑 API Configuration")
    
    # DeepSeek API Key (用于对话)
    deepseek_key = st.text_input(
        "DeepSeek API Key",
        type="password",
        help="用于对话模型的 DeepSeek API Key"
    )
    if deepseek_key:
        st.session_state.deepseek_api_key = deepseek_key

    # 嵌入模型选择
    st.header("🔤 嵌入模型配置")
    embedder_choice = st.selectbox(
        "选择嵌入模型",
        ["Ollama本地模型", "Azure OpenAI", "其他服务"]
    )
    
    if embedder_choice == "Ollama本地模型":
        ollama_url = st.text_input(
            "Ollama Base URL", 
            value="http://localhost:11434",
            help="Ollama 服务地址"
        )
        ollama_model = st.text_input(
            "嵌入模型名称",
            value="nomic-embed-text",
            help="推荐: nomic-embed-text, all-minilm"
        )
        
    elif embedder_choice == "Azure OpenAI":
        azure_api_key = st.text_input("Azure API Key", type="password")
        azure_endpoint = st.text_input("Azure Endpoint")
        azure_embedding_model = st.text_input("嵌入模型", value="text-embedding-ada-002")
        
    elif embedder_choice == "其他服务":
        other_embedder_key = st.text_input("嵌入服务 API Key", type="password")
        other_embedder_url = st.text_input("嵌入服务 Base URL")

    # Qdrant 配置保持不变...
    qdrant_key = st.text_input("Qdrant API Key", type="password")
    qdrant_url = st.text_input("Qdrant URL")

    2. 免费使用
    3. 数据隐私性好
