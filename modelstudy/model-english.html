    大模型相关英语词汇

    寒梦

    # 大模型相关英语词汇

    NLP 包括 NLU 和 NLG。

    1. NLU：Natural Language Understanding，自然语言理解
    2. NLP：Natural Language Processing, 自然语言处理。这是一个更广泛的领域，涵盖了所有让计算机处理、理解、生成人类语言的技术。NLP 包括 NLU 和 NLG。
    3. NLG: Natural Language Generation，自然语言生成。

    简单来说，如果 NLU 是语言的“输入”和理解，那么 NLG 就是语言的“输出”和生成。它的目标是让机器用人类自然语言的形式，生成有意义、连贯且符合语境的文本。

    1. Hallucination: 幻觉
    2. Stop Sequence： 停止序列，个人觉得可以叫： 停止词

    # transformer 相关英语词汇

    1. Multi-H: Multi-Head Attention: 多头注意力
    2. FFN: Feed Forward Network：前馈神经网络
    3. PE: Positional Encoding: 位置编码
    4. LN: Layer Normalization: 层归一化
    5. RC: Residual Connection: 残差连接
    6. Add & Norm: 残差和标准化
    7. Softmax: softmax 函数
    8. Dropout: 随机失活
    9. 缩放点积注意力机制：Scaled Dot-Product Attention
    10. 掩码机制（Masked Attention）
    11. 交叉注意力（Cross-Attention）
    12. 交叉熵损失（Cross-Entropy Loss）
    13. Pre-LN: Pre-Layer Normalization
    14. Post-LN: Post-Layer Normalization
    15. POPE: Rotary Position Embedding, 旋转位置编码
    16. KL: Kullback-Leibler 散度, KL 散度

    # 其他相关

    • RNN：循环神经网络(Recurrent Neural Network)

    • LSTM：长短期记忆网络（LSTM）架构 or 长短期记忆网络（Long Short-Term Memory）

    • NMT，Neural Machine Translation 神经机器翻译

    • BERT，Bidirectional Encoder Representations from Transformers 双向编码器表示从 Transformers

    • GPT，Generative Pre-trained Transformer 生成式预训练 Transformer

    • 缩放点积注意力机制：Scaled Dot-Product Attention

    • 卷积神经网络（CNN）

    • 卷积核（Kernel）

    • RLHF：基于人类反馈的强化学习

    1. Arbitration history：仲裁历史
    2. Arbitration：仲裁
    3. confidence：信心、置信度、得分值
    4. Response time percentiles (approximated): 响应时间百分位数（近似值）

    1. kernel: 内核 jupyter 内核
    2. Thought: 思想
    3. COT：Chain-of-Thought, 思维链
    4. In-Context Learning (ICL): 一段提示词，让模型根据提示词里的指令，去生成回答

    # 标记

    1. Start of Sequence： <SOS>
    2. Beginning of Sequence: <BOS>
    3. 填充掩码（Padding Mask）
    4. 未来信息掩码（Look-ahead Mask）

    # RAG 相关

    1. RAG(Retrieval Augmented Generation)-检索增强⽣成

    将⼤模型应⽤于实际业务场景时会发现，通⽤的基础⼤模型基本⽆法满⾜我们的实际业务需求，主要有以下⼏⽅⾯原因

    • LLM的知识不是实时的，不具备知识更新
    • LLM可能不知道你私有的领域/业务知识
    • LLM有时会在回答中⽣成看似合理但实际上是错误的信息
    1. Chain of Thought: COT 思维链

    # Langchain 框架

    1. LCEL 是 LangChain Expression Language 的缩写，即 LangChain 表达式语言。

    # RAG 相关

    1. Reciprocal Rank Fusion (RRF): 倒数排名融合
    2. Self-Reflective Retrieval-Augmented Generation(Self RAG)
    3. RAG Fusion（RAG 融合）

    1. Vector Search Results：向量搜索结果
    2. Lexical Search results：关键字搜索
    3. Lexical：词汇
    4. Sparse: 稀疏，BGE-M3: Sparse 召回
    5. Dense: 稠密，Qwen3-Embedding 稠密召回

    # 其他

    1. Hallucination 幻觉
    2. SELF-RAG（Self-Reflective Retrieval-Augmented Generation）

    # Fine Tuning

    1. PEFT 的全称是 Parameter-Efficient Fine-Tuning
    2. IA3 的全称是 Infused Adapter by Inhibiting and Amplifying Inner Activations, 3个I， 3个A，所以叫IA3
    3. 稀疏微调（Sparse Fine-tuning）

    # 评估指标相关

    # RAG 检索系统

    1. nDCG (Normalized Discounted Cumulative Gain)：归一化折损累积增益, CG: 累积增益

    RAG场景中的核心作用：

    • 评估检索模块的排序质量（而非仅召回率）：例如在文档检索阶段，判断模型是否将最相关的文档排在前N位（直接影响后续大模型回答的准确性）。
    • 常见于RAG流水线的检索器调优（如向量数据库参数调整、Embedding模型选型）和端到端评估。

    它既考虑返回项的相关性, 也考虑它们在排序列表中的位置。其核心思想是高度相关的项应该更早地出现在列表中。

    1. accuracy 准确性、准确度

    2. f1

    4. exact_match: 精确匹配

    5. semantic: 语义

    6. semantic search: 语义搜索

    7. ground_truth: 基准真值 / 标准答案

    # 评测相关

    1. End-to-End Evaluation: 端到端评测

    # 大模型相关

    1. 在自然语言处理中，"logits" 通常指模型输出的原始预测值（未归一化的概率分数）。
    2. Continuous Batching（连续批处理）

    # 向量数据库

    1. 近似最近邻(ANN: Approximate Nearest Neighbor)
    2. 相似性搜索(Similarity Search)
    3. 倒排索引技术，IVF

    # 倒排索引技术

    倒排索引是一种将“内容（如单词、关键词）”映射到“其所在文档/位置”的索引技术，核心是从“找内容”反向到“找内容在哪”，是搜索引擎、数据库全文检索的核心技术。
    例如在文档中搜索“人工智能”，它能直接返回所有包含该词的文档列表，而非逐篇扫描所有文档。

    # IVF（倒排文件，Inverted File）

    IVF是倒排索引的一种经典存储实现，将倒排索引的核心结构（关键词词典、倒排列表）以文件形式高效存储，是早期信息检索系统中实现倒排索引的主流方式，为快速查询提供了底层数据支撑。

    # 大模型相关

    1. Proximal Policy Optimization(PPO 算法)
    2. critic 模型（价值网络）扮演着重要角色，它主要用于评估策略的好坏，计算优势函数（Advantage Function）
    3. 推理：inference，项目里，也总是会用 infer.py 当做文件名 or 定义 def inference() 函数
    4. Self-consistency: 自我一致性
    5. ground_truth: 真实值
    6. confidence：可信度、置信度

    # 批处理

    1. latency: 延迟
    2. throughput: 吞吐量

    # 多模态

    1. Diffusion：扩散
    2. Stable Diffusion: 稳定扩散
    3. Vision Transformer：视觉编码器、图像编码器
    4. DDPM: Denoising Diffusion Probabilistic Models
    5. multimodal: 多模态，简称：MM, MM-RLHF: 它是大名鼎鼎的 RLHF 在多模态领域的扩展和升级。

    # 影像生成模型

    1. Fréchet Inception Distance：FID，衡量生成图像分布与真实图像分布的相似度。
    2. Peak Signal-to-Noise Ratio：PSNR， 评估生成图像与参考图像（真实图像）的像素级相似度，反映失真程度。

    3. ReAct Prompting

    1. Ragas: Retrieval-Augmented Generation Assessment, 是专为 RAG 系统设计的开源评估框架
    2. context_precision: 上下文精确率
    3. context_recall: 上下文召回率
    4. faithfulness: 忠实度
    5. answer_relevancy: 答案相关性
