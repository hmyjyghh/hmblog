    相关英语词汇

    寒梦

    # 相关英语词汇

    # 大模型相关词汇

    NLP 包括 NLU 和 NLG。

    1. NLU：Natural Language Understanding，自然语言理解
    2. NLP：Natural Language Processing, 自然语言处理。这是一个更广泛的领域，涵盖了所有让计算机处理、理解、生成人类语言的技术。NLP 包括 NLU 和 NLG。
    3. NLG: Natural Language Generation，自然语言生成。

    简单来说，如果 NLU 是语言的“输入”和理解，那么 NLG 就是语言的“输出”和生成。它的目标是让机器用人类自然语言的形式，生成有意义、连贯且符合语境的文本。

    # transformer 相关英语词汇

    1. Multi-H: Multi-Head Attention: 多头注意力
    2. FFN: Feed Forward Network：前馈神经网络
    3. PE: Positional Encoding: 位置编码
    4. LN: Layer Normalization: 层归一化
    5. RC: Residual Connection: 残差连接
    6. Add & Norm: 残差和标准化
    7. Softmax: softmax 函数
    8. Dropout: 随机失活
    9. 缩放点积注意力机制：Scaled Dot-Product Attention
    10. 掩码机制（Masked Attention）
    11. 交叉注意力（Cross-Attention）
    12. 交叉熵损失（Cross-Entropy Loss）
    13. Pre-LN: Pre-Layer Normalization
    14. Post-LN: Post-Layer Normalization
    15. POPE: Rotary Position Embedding, 旋转位置编码

    # 其他相关

    • RNN：循环神经网络(Recurrent Neural Network)

    • LSTM：长短期记忆网络（LSTM）架构 or 长短期记忆网络（Long Short-Term Memory）

    • NMT，Neural Machine Translation 神经机器翻译

    • BERT，Bidirectional Encoder Representations from Transformers 双向编码器表示从 Transformers

    • GPT，Generative Pre-trained Transformer 生成式预训练 Transformer

    • 缩放点积注意力机制：Scaled Dot-Product Attention

    • 卷积神经网络（CNN）

    • 卷积核（Kernel）

    • RLHF：基于人类反馈的强化学习

    # RAG 相关

    1. RAG(Retrieval Augmented Generation)-检索增强⽣成

    将⼤模型应⽤于实际业务场景时会发现，通⽤的基础⼤模型基本⽆法满⾜我们的实际业务需求，主要有以下⼏⽅⾯原因

    • LLM的知识不是实时的，不具备知识更新
    • LLM可能不知道你私有的领域/业务知识
    • LLM有时会在回答中⽣成看似合理但实际上是错误的信息
    1. Chain of Thought: COT 思维链

    # Langchain 框架

    1. LCEL 是 LangChain Expression Language 的缩写，即 LangChain 表达式语言。

    # RAG 相关

    1. Reciprocal Rank Fusion (RRF): 使用 Reciprocal Rank Fusion (RRF) 算法合并所有检索结果, 结果融合

    Reciprocal Rank Fusion (RRF) : 倒数秩序融合

    1. Self-Reflective Retrieval-Augmented Generation(Self RAG)

    # 核心思想

    RRF 的核心思想是：一个文档在多个排名列表中的位置都很靠前，那么它本身就应该是一个非常相关、非常重要的文档。

    Reciprocal_rank_fusion that takes multiple lists of ranked documents 
and an optional parameter k used in the RRF formula

Reciprocal_rank_fusion，接受多个排名文档列表, 以及RRF公式中使用的可选参数k
    1. RAG Fusion（RAG 融合）

    # Fine Tuning

    1. PEFT 的全称是 Parameter-Efficient Fine-Tuning
    2. IA3 的全称是 Infused Adapter by Inhibiting and Amplifying Inner Activations, 3个I， 3个A，所以叫IA3
    3. 稀疏微调（Sparse Fine-tuning）

    # 评估指标相关

    # Evaluate

    1. accuracy 准确性、准确度
    2. f1
    3. metric: 指标
    4. exact_match: 精确匹配

    # 大模型相关

    1. 在自然语言处理中，"logits" 通常指模型输出的原始预测值（未归一化的概率分数）。
    2. Continuous Batching（连续批处理）

    # 向量数据库

    1. 近似最近邻(ANN: Approximate Nearest Neighbor)
    2. 相似性搜索(Similarity Search)

    # 大模型相关

    1. Proximal Policy Optimization(PPO 算法)
    2. critic 模型（价值网络）扮演着重要角色，它主要用于评估策略的好坏，计算优势函数（Advantage Function）

    # 批处理

    1. latency: 延迟
    2. throughput: 吞吐量

    # 多模态

    1. Diffusion：扩散
    2. Stable Diffusion: 稳定扩散
    3. Vision Transformer：视觉编码器、图像编码器
    4. DDPM: Denoising Diffusion Probabilistic Models
    5. multimodal: 多模态，简称：MM, MM-RLHF: 它是大名鼎鼎的 RLHF 在多模态领域的扩展和升级。

    # 影像生成模型

    1. Fréchet Inception Distance：FID，衡量生成图像分布与真实图像分布的相似度。
    2. Peak Signal-to-Noise Ratio：PSNR， 评估生成图像与参考图像（真实图像）的像素级相似度，反映失真程度。
