    # DataLoader 的作用

    DataLoader 是 PyTorch 中用于数据加载的核心工具，主要作用是将数据集（Dataset）按指定方式封装成可迭代的批次数据，方便模型训练或推理时高效读取和处理数据。

    它的核心功能和优势体现在以下几个方面：

    1. 批次处理数据
      将数据集按 batch_size 拆分成多个小批次（batch）。例如设置 batch_size=32 时，每次迭代会返回 32 个样本，避免一次性加载全部数据导致内存溢出，同时也符合神经网络的批量计算特性。

    2. 数据打乱（shuffle）
      通过 shuffle=True 可以在每个 epoch（完整遍历一次数据集）前随机打乱数据顺序，减少数据顺序对模型训练的影响，提高模型泛化能力。如果是测试/验证阶段，通常设为 shuffle=False 保持数据顺序稳定。

    3. 并行加速
      支持通过 num_workers 参数设置多进程加载数据（默认值为 0，即主进程加载），在数据预处理耗时较长时（如图片解码、增强等），能显著提升数据加载速度，避免 GPU 等待数据而空闲。

    4. 自动拼接与转换
      自动将批次数据拼接成张量（Tensor），并可配合 collate_fn 参数自定义批次数据的拼接逻辑（例如处理变长序列、不同结构的数据）。

    5. 迭代器接口
      封装成可迭代对象，方便在训练循环中使用：

    for batch_idx, (data, label) in enumerate(dataloader):
  # data: 批次数据（张量），shape 为 (batch_size, ...)
  # label: 批次标签（张量）
  outputs = model(data)
  # 计算损失、反向传播等

    总结来说，DataLoader 是连接数据集和模型的桥梁，通过高效的数据分批、打乱、并行加载等机制，优化了数据流向模型的整个过程，是 PyTorch 训练流程中不可或缺的组件。
