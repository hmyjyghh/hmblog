    # React Native 简史

    # React Native的兴起

    React Native是React的一个重要衍生项目，它允许开发者使用JavaScript和React来开发原生移动应用。React Native的兴起进一步扩展了React的应用场景，使得开发者可以使用一套代码同时开发Web和移动应用。 从2018年开始，React Native团队开始重写底层实现，推出了New Architecture。这个新架构包括JSI、Fabric、Turbo Modules、Codegen和Hermes Engine等核心组件，极大地提高了React Native的性能和稳定性。从React Native 0.76开始，New Architecture将成为默认模式。
