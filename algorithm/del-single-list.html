    1. 删除单链表倒数第K个节点

    寒梦

    # 1. 删除单链表倒数第K个节点

    • 通过两个指针来实现,其中一个指针比另一个指针提前K个节点。
    • 这样,当前面的指针到达链表末尾时,后面的指针正好位于需要删除的节点的前一个位置。
    # 定义链表节点的类
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val      # 节点存储的值
        self.next = next    # 指向下一个节点的指针

class Solution:
    def removeWithFromEnd(self, head: ListNode, n: int) -> ListNode:
        # 创建一个虚拟头节点，用于处理删除头节点的特殊情况
        dummy = ListNode(0)
        # 将虚拟头节点连接到原链表的头部
        dummy.next = head
        # 初始化两个指针，都指向虚拟头节点
        first = dummy
        second = dummy

        # 将第一个指针向前移动n+1个位置，使两个指针之间保持n个节点的间隔
        for __ in range(n + 1):
            first = first.next

        # 将两个指针同时向前移动，直到第一个指针到达链表末尾
        while first:
            first = first.next      # 第一个指针向前移动
            second = second.next    # 第二个指针向前移动

        # 此时第二个指针指向要删除节点的前一个节点
        # 通过修改指针来删除目标节点
        second.next = second.next.next

        # 返回虚拟头节点的下一个节点（即新链表的头节点）
        return dummy.next

    总结一下,通过使用两个指针,一个比另一个提前K个节点,实现了删除单链表倒数第K个节点。时间复杂度为O(n),因为我们需要遍历链表一次。空间复杂度为O(1),因为我们只使用了常数级别的额外空间。

    # 2. 删除单链表中等于给定值val的所有节点

    样例:给出链表1->2->3->3->4->5->3,val=3,你需要返回删除3之后的链表:1->2->4->5
这个问题可以通过遍历链表,并检查每个节点的值是否等于val来解决。为了简化边界情况处理,我们使用一个虚
拟头节点。

    # 定义链表节点的类
class ListNode:
    def __init__(self, val=0, next=None):
    self.val = val
    self.next = next

class Solution:
    def removeElements(self, head: ListNode, val: int) -> ListNode:
    # 创建一个虚拟头节点，以处理边界情况
    dummy = ListNode(0)
    dummy.next = head
    current = dummy

# 遍历链表
while current.next:
    if current.next.val == val:
    # 如果下一个节点的值等于给定值，跳过该节点
    current.next = current.next.next
    else:
    # 否则，移动到下一个节点
    current = current.next

# 返回头节点
return dummy.next

    # 3. 使用链表实现栈和队列

    # 3.1 使用链表实现栈

    • 栈的特点是后进先出(LIFO),我们可以在链表的头部插入和删除节点来买现这一点
    # 定义链表节点的类
class ListNode:
    def __init__(self, val=0, next=None):
    self.val = val
    self.next = next

class Stack:
    def __init__(self):
    # 初始化栈为空
    self.top = None

    def push(self, x: int) -> None:
    # 在栈顶插入新节点
    node = ListNode(x)
    node.next = self.top
    self.top = node

    def pop(self) -> int:
    # 从栈顶删除节点，并返回其值
    if self.top is None:
    raise IndexError("pop from empty stack")
    value = self.top.val
    self.top = self.top.next
    return value

    def peek(self) -> int:
    # 返回栈顶节点的值
    if self.top is None:
    raise IndexError("peek from empty stack")
    return self.top.val

    def empty(self) -> bool:
    # 判断栈是否为空
    return self.top is None

    # 3.2 使用链表实现队列

    • 队列的特点是先进先出(FIFO),我们可以在链表的头部删除节点, 在链表的尾部插入节点来实现这一点。

    • 头部是指排队队伍的最前面

    class Queue:
    def __init__(self):
    # 初始化队列为空
    self.front = None
    self.rear = None
# 入队
def enqueue(self, x: int) -> None:
    # 在队列尾部插入新节点
    node = ListNode(x)
    if self.rear is None:
    self.front = self.rear = node
    else:
    self.rear.next = node
    self.rear = node

# 出队
def dequeue(self) -> int:
    # 从队列头部删除节点，并返回其值
    if self.front is None:
    raise IndexError("dequeue from empty queue")
    value = self.front.val
    self.front = self.front.next
    if self.front is None:
    self.rear = None
    return value

def peek(self) -> int:
    # 返回队列头部节点的值
    if self.front is None:
    raise IndexError("peek from empty queue")
    return self.front.val

def empty(self) -> bool:
    # 判断队列是否为空
    return self.front is None

    # 4. 打印两个有序链表的公共部分

    • 通过同时遍历两个链表,并比较每个节点的值来解决。
    # 定义链表节点的类
class ListNode:
    def __init__(self, val=0, next=None):
    self.val = val
    self.next = next

class Solution:
    def printCommonParts(self, head1: ListNode, head2: ListNode) -> None:
    # 初始化两个指针，分别指向两个链表的头节点
    current1 = head1
    current2 = head2

# 遍历两个链表，直到其中一个为空
while current1 is not None and current2 is not None:
    if current1.val < current2.val:
    # 如果链表1的当前节点值小于链表2的当前节点值，移动链表1的指针
    current1 = current1.next
    elif current1.val > current2.val:
    # 如果链表1的当前节点值大于链表2的当前节点值，移动链表2的指针
    current2 = current2.next
else:
    # 如果两个链表的当前节点值相等，打印这个值并移动两个指针
    print(current1.val, end=' ')
    current1 = current1.next
    current2 = current2.next

    # 5. 判断链表是否有环

    • 这个问题可以通过快慢指针来解决。
    • 快指针每次移动两个节点,慢指针每次移动一个节点。
    • 如果链表中存在环,快指针最终会与慢指针相遇。
    # 定义链表节点的类
class ListNode:
    def __init__(self, val=0, next=None):
    self.val = val
    self.next = next

class Solution:
    def hasCycle(self, head: ListNode) -> bool:
    # 初始化快慢指针
    slow = head
    fast = head

# 遍历链表
while fast is not None and fast.next is not None:
    slow = slow.next    # 慢指针每次移动一个节点
    fast = fast.next.next    # 快指针每次移动两个节点

# 如果快慢指针相遇，说明链表中存在环
    if slow == fast:
    return True

# 如果遍历结束快指针没有相遇慢指针，说明链表中没有环
return False

    # 6. 如果链表有环,返回链表中环的起始节点

    # 定义链表节点的类
class ListNode:
    def __init__(self, val=0, next=None):
    self.val = val
    self.next = next

class Solution:
    def detectcycle(self, head: ListNode) -> ListNode:
    # 初始化快慢指针
    slow = head
    fast = head

# 遍历链表，检测是否有环
while fast is not None and fast.next is not None:
    slow = slow.next    # 慢指针每次移动一个节点
    fast = fast.next.next    # 快指针每次移动两个节点

# 如果快慢指针相遇，说明链表中存在环
if slow == fast:
    # 通过快慢指针相遇点来找到环的起始节点
    # 将慢指针移动到链表头部
    slow = head

# 快慢指针同时移动一个节点，直到它们相遇
while slow != fast:
    slow = slow.next
    fast = fast.next

# 相遇点即为环的起始节点
return slow

# 如果遍历结束快指针没有相遇慢指针，说明链表中没有环
return None

    # 7. 删除链表中间节点

    • 这个问题可以通过使用快慢指针来找到中间节点,然后将其册滁。
    # 定义链表节点的类
class ListNode:
    def __init__(self, val=0, next=None):
    self.val = val
    self.next = next

class Solution:
    def deleteMiddle(self, head: ListNode) -> ListNode:
    # 如果链表为空或只有一个节点，返回None
    if not head or not head.next:
    return None

    # 初始化快慢指针和前置指针
    slow = head
    fast = head
    prev = None

    # 遍历链表，找到中间节点
    while fast and fast.next:
    prev = slow
    slow = slow.next
    fast = fast.next.next

    # 删除中间节点
    prev.next = slow.next

    return head
    • 总结一下,这个算法通过使用快慢指针来找到链表的中间节点,然后后将其删除
    • 时间复杂度为O(n),因为我们需要遍历链表一次。
    • 空间复杂度为O(1),因为我们只使用了常数级别的额外空间。
