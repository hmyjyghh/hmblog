    1. 实现一个带有取最小值min 方法的栈

    寒梦

    # 1. 实现一个带有取最小值min 方法的栈

    • 实现一个带有取最小值min方法的栈,min方法返回当前栈中的最小值

      • 实现的栈将支持push,pop和min操作,所有操作要求都在O(1)时间内完成
      • 我们可以使用两个栈:一个用于存储所有元素,另一个用于存当前最小值
    class MinStack:
    def __init__(self):
        # 初始化两个栈，一个用于存储所有元素，另一个用于存储最小值
        self.stack = []
        self.min_stack = []
    
    def push(self, x:int) -> None:
        # 将元素x推入主栈
        self.stack.append(x)
        # 如果最小值栈为空，或x小于等于当前最小值，则将x推入最小值栈
        # -1 是指栈顶: 先进后出，相比队列，尾部就是顶部,也就是栈顶
        if not self.min_stack or x <= self.min_stack[-1]:
            self.min_stack.append(x)
    
    def pop(self) -> None:
        # 从主栈中弹出元素
        if self.stack:
            top = self.stack.pop()
            # 如果弹出的元素等于最小值栈的栈顶元素，则从最小值栈中也弹出该元素
            if top == self.min_stack[-1]:
                self.min_stack.pop()
    
    def top(self) -> int:
        # 返回主栈的栈顶元素
        if self.stack:
            return self.stack[-1]
        raise IndexError("Stack is empty")
    

    def getMin(self) -> int:
        # 返回最小值栈的栈顶元素，即当前栈中的最小值
        if self.min_stack:
            return self.min_stack[-1]
        raise IndexError("MinStack is empty")

    总结一下,这个带有取最小值min方法的栈通过使用`两个栈`来实现所有操作都在O(1)时间内完成。
主栈用于存储所有元素,最小值栈用于存储当前最小值。每当有新元素入栈时,如果该元素小于或等于当前最小
值,则将其同时推入最小值栈;
当元素出栈时,如果该元素等于当前最小值,则从最小值栈中也弹出该元素。
- 这样可以确保我们在O(1)时间内完成push,pop和min操作。

    # 2. 使用数组实现一个栈,需要提供自动扩容和缩容功能

    • 我们将定义一个栈类,内部维护一个数组用于存储元素,并在需要时调整数组的大小。
    class ArrayStack:
    def __init__(self, capacity=2):
        # 初始化栈，初始容量为2
        self.stack = [None] * capacity
        self.capacity = capacity
        self.size = 0
    
    def push(self, x):
        # 如果栈已满，则进行扩容
        if self.size == self.capacity:
            self._resize(2 * self.capacity)
        # 将元素x推入栈中
        self.stack[self.size] = x
        self.size += 1
    
    def pop(self):
        if self.size == 0:
            raise IndexError("pop from empty stack")
        # 从栈中弹出元素
        x = self.stack[self.size - 1]
        self.stack[self.size - 1] = None
        self.size -= 1
        # 如果栈的大小小于容量的四分之一，则进行缩容(感觉应该是扩容吧---待确定)
        if 0 < self.size == self.capacity // 4:
            self._resize(self.capacity // 2)
        return x
    

    def peek(self):
        if self.size == 0
            raise IndexError("pop from empty stack")
        # 返回栈顶元素
        return self.stack[self.size - 1]

    def _resize(self, new_capacity):
        # 调整数组大小
        new_stack = [None] * new_capacity
        for i in range(self.size):
            new_stack[i] = self.stack[i]
        self.stack = new_stack
        self.capacity = new_capacity
    
    def __len__(self):
        # 返回当前栈的大小
        return self.size
    
    def is_empty(self):
        # 判断栈是否为空
        return self.size == 0

    总结一下,这个使用数组实现的栈通过在需要时调整数组大小小来实现自动扩容和缩容功能。当栈满时,数组容量加
倍;当栈大小小于容量的四分之一时,数组容量减半。时间复杂度为00(1)均摊,因为调整大小的操作在多次push
和pop操作中摊销开来。

    # 3. 两个队列实现栈

    • 通过两个队列来实现栈的后进先出(LIFO)特性。

    • 具体方法是将新元素总是加入到一个空队列中, 然后将另一个队列的所有元素依次移到这个队列中,从而实现栈的效果。

    from collections import deque

class MyStack:
    def __init__(self):
        # 初始化两个队列
        self.quene1 = deque()
        self.quene2 = deque()
    
    def push(self, x: int) -> None:
        # 将新元素加入到空的quene2 中
        self.quene2.append(x)
        # 将quene1的所有元素移动quene2 中

        while self.quene1:
            self.quene2.append(self.quene1.popleft())
        # 交换quene1 和 quene2 
        self.quene1, self.quene2 = self.quene2, self.quene1
    
    def pop(self) -> int:
        # 从quene1 中弹出元素，即为栈顶元素
        if self.quene1:
            return self.quene1.popleft()
        else:
            raise IndexError("pop from empty stack")
    
    def top(self) -> int:
        # 返回quene1 的队首元素，即为栈顶元素
        if self.quene1：
            return self.quene1[0]
        else:
            raise IndexError("top from empty stack")
    
    def empty(self) -> bool:
        # 判断栈是否为空
        return not self.quene1

    总结一下,通过两个队列实现栈的算法,通过在一个空队列中加入新元素, 并将另一个队列的所有元素依次移到这个队列中,从而实现栈的后进先出(LIFO)特性。

    push操作的时间复杂度为O(n),因为需要将所有元素移动到新队列中中,而pop和top操作的时间复杂度为0(1)。

    # 4. 字符串查找

    给定的source字符串和target字符串,在source字符串中找出target字符串出现的第一个位置(从0开始)。若 不存在,返回-1 cool968(59026571) 样例: 如果source="source"和target="target",返回-1 如果source="abcdabcdefg"和target="bcd",返回1 这个问题可以通过暴力匹配或KMP来解决,在面试时候,至少要能能写出暴力匹配算法,它的时间复杂度是 O(n*m),其中n是source的长度,m是target的长度。

    class Solution:
    def strStr(self, source: str, target:str) -> int:
        # 获取source 和 target 的长度
        n, m = len(source), len(target)

        # 边界条件, 如果target为空字符串， 返回0
        if m == 0:
            return 0
        # 遍历source 字符串
        for i in in range(n - m + 1):
            # 检查从当前索引开始的子字符串是否与target 匹配
            if source[i:i + m] == target:
                return i
        
        # 如果遍历结束后没有匹配到，返回-1
        return -1

    # 5. 逆序一个栈

    我们可以使用递归的方法。需要两个步骤:首先,通过递归的的方法将栈顶元素取出,然后递归逆序剩余的栈。其 次,将取出的元素插入到逆序后的栈底。

    class Solution:
    def reverseStack(self, stack):
        # 如果栈为空，直接返回, (not stack 没有栈，栈为空)
        if not stack:
            return
        
        # 递归地取出栈顶元素
        top = stack.pop()
        self.reverseStack(stack)

        # 将取出的元素插入到逆序后的栈底
        self.insertAtBottom(stack, top)
    
    def insertAtBottom(self, stack, value):
        # 如果栈为空，直接将值推入栈中
        if not stack:
            stack.append(value)
            return
        
        # 递归地取出栈顶元素
        top = stack.pop()
        self.insertAtBottom(stack, value)

        # 将取出的元素重新推入栈中
        stack.append(top)

    总结一下,算法时间复杂度为O(n2)

    1.reverseStack函数: 。每次递归调用处理栈顶元素,将问题规模减1(即处理n-1的栈)。 每次递归返回后调用 insertAtBottom,将当前元素插入到已反转的栈底。 2.insertAtBottom函数: 需要将栈中所有元素依次弹出再重新压入,时间复杂度为O(k),其中k是当前找的大小。 3.总时间复杂度: 对于规模为n的栈, reversestack 递归n次,每次递归调用 insertAtBottom。 每次调用insertAtBottom的时间复杂度依次为0,1,2,.........,n-1 总操作次数为:0+1+2+...+(n-1)=n(n-1)/2→O(n2)。 空间复杂度为O(n),因为递归调用栈的深度为n,其中n是栈的大小。

    # 6. 用一个栈实现对另一个栈的排序

    通过一个辅助栈来对另一个找进行排序。具体来说,我们将使用一个临时找来帮助排序原栈中的元素。这个算法的 思想是将原栈中的元素逐个弹出,并按顺序插入到临时找中,从而实现排序。 以下是具体实现:

    class Solution:
    def sortStack(self, stack):
        # 创建一个临时栈用于排序
        temp_stack =[]

        # 遍历原栈中的每个元素
        while stack:
            # 取出原栈的栈顶元素
            current = stack.pop()

            # 将临时栈中大于当前元素的元素依次弹出并推入原栈
            while temp_stack and temp_stack[-1] > current:
                stack.append(temp_stack.pop())
            
            # 将当前元素推入临时栈
            temp_stack.append(current)
        
        # 将临时栈中的元素重新推入原栈，完成排序
        while temp_stack:
            stack.append(temp_stack.pop())

# 示例代码
solution = new Solution()
stack = [3, 1, 4, 2, 5]
solution.sortStack(stack)
print(stack) # 输出: [1, 2, 3, 4, 5]
