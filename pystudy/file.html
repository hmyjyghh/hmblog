    # 基本数据类型

    1. 数字（Number）
    2. 字符串（String）
    3. 列表（List）
    4. 元组（Tuple）
    5. 字典（Dictionary）
    6. 集合（Set）
    7. 布尔值（Boolean）
    8. 空值（None）

    列表（List） list = ['123', 46, 'str'];

    元组（Tuple） tuple = ('123', 46, 'str');

    字典（Dictionary） dict = {'name': '张三', 'age': 18};

    元组的元素不能修改

    List 和 元组 都是 里面的元素类型可以不一样

    Set 无序，唯一，

    字典：它是一个无序的 键(key) : 值(value) 的集合。
