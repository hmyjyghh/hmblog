    format() 函数的作用

    寒梦

    # format() 函数的作用

    这里调用 format 函数是为了字符串格式化，具体来说是使用 字符串的 format() 方法 来将变量值插入到字符串模板中。

    # format() 函数的作用

    # 1. 动态插入变量值

    format() 方法用花括号 {} 作为占位符，然后将参数按顺序替换到这些位置：

    name = "Alice"
age = 25
message = "我叫{}，今年{}岁。".format(name, age)
print(message)  # 输出: 我叫Alice，今年25岁。

    # 2. 在你的代码中的具体应用

    return """你是一个专业的数据科学家助手。...
        当前数据集信息：
        - 数据集路径: {}
        - 数据形状: {}
        - 列名: {}
        - 前5行数据预览: {}
        """.format(
            self.csv_path,        # 替换第一个 {}
            self.df.shape,        # 替换第二个 {}
            list(self.df.columns), # 替换第三个 {}
            self.df.head().to_string() # 替换第四个 {}
        )

    执行后效果相当于：

    return f"""你是一个专业的数据科学家助手。...
        当前数据集信息：
        - 数据集路径: /path/to/titanic.csv
        - 数据形状: (891, 12)
        - 列名: ['PassengerId', 'Survived', 'Pclass', 'Name', 'Sex', 'Age', ...]
        - 前5行数据预览:    PassengerId  Survived  Pclass  ...  
0            1         0       3  ...
1            2         1       1  ...
...
        """

    # 为什么使用 format() 而不是其他方法？

    # 方法比较：

    # 1. 使用 format()（你代码中的方式）

    text = "数据集路径: {}, 形状: {}".format(path, shape)

    # 2. 使用 f-string（Python 3.6+ 推荐）

    text = f"数据集路径: {path}, 形状: {shape}"

    # 3. 使用 % 格式化（旧方式）

    text = "数据集路径: %s, 形状: %s" % (path, shape)

    # 4. 字符串拼接（不推荐）

    text = "数据集路径: " + path + ", 形状: " + str(shape)

    # 在你的场景中为什么这样设计？

    # 1. 动态生成系统提示词

    def _get_system_prompt(self) -> str:
    return """...{}...""".format(
        self.csv_path,          # 动态插入文件路径
        self.df.shape,          # 动态插入数据形状
        list(self.df.columns),  # 动态插入列名
        self.df.head().to_string()  # 动态插入数据预览
    )

    # 2. 让Agent了解当前数据状态

    通过动态插入数据信息，Agent在每次初始化时都能获得：

    • 具体的数据文件路径
    • 实时的数据形状（行数×列数）
    • 准确的列名列表
    • 实际的数据内容预览

    这样Agent就能基于真实数据来回答问题，而不是基于固定的模板。

    # 实际效果示例

    假设你的数据是：

    • self.csv_path = "titanic.csv"
    • self.df.shape = (891, 12)
    • list(self.df.columns) = ['PassengerId', 'Survived', 'Pclass', ...]

    格式化后的提示词将是：

    你是一个专业的数据科学家助手...
当前数据集信息：
- 数据集路径: titanic.csv
- 数据形状: (891, 12)
- 列名: ['PassengerId', 'Survived', 'Pclass', 'Name', 'Sex', 'Age', ...]
- 前5行数据预览: (实际的数据表格)

    # 现代写法：使用 f-string

    如果是 Python 3.6+，更推荐使用 f-string，更简洁：

    def _get_system_prompt(self) -> str:
    return f"""你是一个专业的数据科学家助手。你擅长数据探索、统计分析和机器学习。
        你可以帮助用户：
        1. 提供数据的统计摘要（均值、方差、最大最小值等）
        2. 创建数据可视化图表
        3. 训练机器学习模型并进行预测

        当前数据集信息：
        - 数据集路径: {self.csv_path}
        - 数据形状: {self.df.shape}
        - 列名: {list(self.df.columns)}
        - 前5行数据预览: {self.df.head().to_string()}

        请用专业但易懂的方式回答用户问题。如果用户的要求不明确，请询问澄清。
        """

    # 总结

    format() 函数在这里的作用是：创建一个包含实时数据信息的动态提示词，让AI Agent能够基于具体的数据上下文来回答问题，而不是使用固定的、通用的提示词。这样可以显著提高Agent的准确性和实用性。
