    # 常见的mcp 工具

    1. 高德导航、qq音乐、查天气

    # 生成文档类的 MCP 工具

    生成文档类的 MCP（Model Context Protocol）工具已经有不少，涵盖了从代码文档、笔记、技术文档到API文档等多种类型。以下是主要的 MCP 工具分类和具体实例：

    # 一、代码文档生成类 MCP 工具

    # 1. Docstring 生成工具

    • mcp-server-pydoc: 为 Python 代码生成和维护 Google/Numpy 风格的 docstring
    • mcp-server-jsdoc: 为 JavaScript/TypeScript 代码生成 JSDoc 注释
    • mcp-server-javadoc: 为 Java 代码生成 JavaDoc 注释

    # 2. 代码文档提取器

    • mcp-server-sphinx: 从代码中提取文档并生成 Sphinx 格式的文档
    • mcp-server-doxygen: 支持从多种语言生成 Doxygen 格式文档

    # 二、笔记和知识库类

    # 3. Markdown 文档工具

    • mcp-server-markdown: 管理和转换 Markdown 文档，支持表格、图表等
    • mcp-server-obsidian: 与 Obsidian 笔记应用集成，管理个人知识库
    • mcp-server-notion: 连接 Notion 数据库，生成和更新文档

    # 4. 文档聚合器

    • mcp-server-knowledge: 从多个来源聚合信息生成综合文档
    • mcp-server-confluence: 与 Atlassian Confluence 集成

    # 三、API 文档生成类

    # 5. OpenAPI/Swagger 工具

    • mcp-server-openapi: 从代码注释自动生成 OpenAPI 规范文档
    • mcp-server-swagger-ui: 生成交互式 API 文档界面
    • mcp-server-postman: 生成 Postman 集合文档

    # 6. API 文档维护

    • mcp-server-api-blueprint: 支持 API Blueprint 格式
    • mcp-server-raml: 支持 RAML 格式的 API 文档

    # 四、技术文档和手册

    # 7. 技术写作助手

    • mcp-server-techdocs: 生成技术文档、用户手册
    • mcp-server-readthedocs: 与 ReadTheDocs 平台集成
    • mcp-server-gitbook: 生成 GitBook 风格文档

    # 8. 架构文档

    • mcp-server-diagrams: 生成架构图文档（支持 PlantUML、Mermaid 等）
    • mcp-server-c4: 生成 C4 模型架构文档

    # 五、文档处理和转换类

    # 9. 格式转换工具

    • mcp-server-pandoc: 文档格式转换（Markdown ↔ HTML ↔ PDF ↔ Word 等）
    • mcp-server-office: 处理 Office 文档（Word、Excel、PowerPoint）

    # 10. 文档分析工具

    • mcp-server-doc-analyzer: 分析文档结构、提取关键信息
    • mcp-server-summarizer: 文档摘要生成

    # 六、特定领域文档

    # 11. 学术文档

    • mcp-server-latex: 生成 LaTeX 学术论文和报告
    • mcp-server-bibtex: 管理参考文献和引用

    # 12. 法律和合规文档

    • mcp-server-legal: 生成法律合同和合规文档模板
    • mcp-server-compliance: 合规文档生成和检查

    # 13. 商业文档

    • mcp-server-business: 生成商业计划书、报告
    • mcp-server-presentation: 生成演示文稿内容和结构

    # 七、文档质量检查类

    # 14. 文档质量工具

    • mcp-server-grammar: 文档语法和风格检查
    • mcp-server-readability: 可读性分析
    • mcp-server-consistency: 检查文档一致性（术语、格式等）

    # 15. 本地化工具

    • mcp-server-i18n: 多语言文档生成和翻译管理
    • mcp-server-localization: 文档本地化支持

    # 八、实用 MCP 服务器推荐

    # 实际可用的 MCP 文档工具：

    1. mcp-server-filesystem (官方)

      • 虽然不是专门的文档工具，但可以读写各种文档文件
      • 支持 Markdown、JSON、YAML、文本文件等

    2. mcp-server-markdown

      • 专门处理 Markdown 文档
      • 支持表格、代码块、链接检查等功能

    3. mcp-server-http (官方)

      • 可以获取网页内容，用于文档研究
      • 支持 API 调用获取文档数据

    4. mcp-server-sqlite

      • 可以管理文档数据库
      • 适合文档元数据和索引管理

    5. mcp-server-github

      • 直接访问 GitHub 上的文档仓库
      • 读写 issues、wiki、文档文件

    # 九、如何查找更多 MCP 工具

    # 1. 官方资源

    # 访问 MCP 官方 GitHub
https://github.com/modelcontextprotocol

# 浏览官方服务器列表
https://github.com/modelcontextprotocol/servers

    # 2. 社区资源

    • Awesome MCP: GitHub 上的 MCP 工具合集
    • NPM Registry: 搜索 mcp-server-* 包
    • PyPI: 搜索 mcp-server-* Python 包

    # 3. 自己创建 MCP 服务器

    如果你需要特定的文档功能，可以创建自己的 MCP 服务器。基本步骤：

    # 示例：创建一个简单的文档生成服务器
from mcp.server import Server
from mcp.server.models import Tool

app = Server("document-generator")

@app.list_tools()
async def handle_list_tools():
    return [
        Tool(
            name="generate_document",
            description="Generate a structured document",
            inputSchema={
                "type": "object",
                "properties": {
                    "template": {"type": "string"},
                    "content": {"type": "string"}
                }
            }
        )
    ]

    # 十、使用建议

    # 1. 开发文档场景

    • 组合使用 filesystem + markdown + diagrams 工具
    • 实现：代码文档 + 架构图 + 用户手册的一站式生成

    # 2. API 文档场景

    • 使用 openapi + postman + http 工具
    • 自动生成并测试 API 文档

    # 3. 知识管理场景

    • 组合 notion + obsidian + summarizer 工具
    • 实现知识的收集、整理和总结

    # 注意要点

    1. MCP 生态系统仍在快速发展，新的工具不断涌现
    2. 许多工具是社区开发的，质量参差不齐，建议查看 GitHub stars 和活跃度
    3. 实际部署时，需要考虑安全性和权限控制
    4. 文档类工具通常需要结合使用才能发挥最大效果
